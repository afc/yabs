# Summary

yabs is quick, panic, dirty R5RS Scheme i12n.

Only with the very basic R5Rs feature, without garbage collection

- fixnum
- boolean
- character
- string
- '()
- pair
- symbol
- quote '
- environment
- if
- some primitive procedures
- simplified lambda i12n
- begin
- cond
- let
- and or not
- apply
- eval
- I/O support
